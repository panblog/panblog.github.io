Hints of swirl

发表于 2017-05-17 | 分类于 Programming ， R |

The swirl R package is a tool to guide user into R and makes it fun and easy to learn R programming and data science. If you are new to R, have no fear.

Two useful swirl websites

  1. http://swirlstats.com/scn/
  2. https://github.com/swirldev/swirl_courses

Manual course installation method

  1. Download course file.
  2. Run swirl::install_course() in the R console.
  3. Select the file you just downloaded.
