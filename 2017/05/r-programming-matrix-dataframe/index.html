Loop functions

Both lapply() and sapply() take a list as input, apply a function to each element of the list, then combine and return the result. lapply() always returns a list, whereas sapply() attempts to simplify the result to return matrix/vector.

lapply

Call the lapply() function with two arguments. The first argument is the object over which we are looping (i.e. flag_colors), and the second argument is the name of the function we wish to apply to each column (i.e. sum). Remember that the second argument is just the name of the function with no parentheses, etc.

sapply

sapply() instead returns a matrix when each element of the list returned by lapply() is a vector of the same length. sapply() will returns a vector if each element of the list which lapply() returns is of length one. If the list returned by lapply() is a vector of different length, sapply() will returns as same as lapply().

vapply

vapply() is a safer variant of sapply(), since it requires you to specify the format of the output in advance. In addition, vapply() may perform faster than sapply() for large datasets. In short, vapply() is safer and faster, and less convenient (when working interactively at the prompt).

tapply

tapply() allows to split data up into groups based on the value of some variable.
