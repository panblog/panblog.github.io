自定义hexo NexT主题

发表于 2017-05-20 | 分类于 Programming ， GitHub |

虽然Hexo的主题已经很完善了，但是对于普通人来说，还是有一些可以优化的地方（虽然对于Geek不是问题）。罗列几个常见的需要解决的问题，供大家参考，欢迎讨论添加。

更改footer

更改链接信息 \themes\next\layout\_partials\footer.swig

更改显示信息 \themes\next\languages\zh-Hans.yml

字体加速

依据以往的经验，如果是使用Google字体库加载速度会非常慢，所以通常的解决方式将//fonts.googleapis.com替换为360的字体加速服//fonts.useso.com。5.1.1版本的NexT对于修改字体库外链地址非常方便，在主题配置文件可以直接修改。
但是根据实测，Google字体库貌似并不慢，知乎上也有看到消息Google Fonts已经被解封，360字体加速服务已经停止。Google Fonts解封不知真假，不过360字体真的不管用了。安全起见，将外链地址改为了中科大的//fonts.lug.ustc.edu.cn，亲测有效。

设置logo

通过网站favicon在线制作制作favicon图片，logo最好设置32*32，将图片放在hexo-site/source/目录下。

CNAME文件丢失导致的主页404

对于购买了域名的用户来说，重新部署后会删除根目录的CNAME文件，导致主页出现404错误。解决方法是，将CNAME、README、favicon.ico文件都放在Source文件夹下，每次上传就不会消失了。
