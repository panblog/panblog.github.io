先简单介绍一下通常的检索后台设置，再单独说一下如何解决百度检索问题（有彩蛋）。

让百度检索你的GitHub站点

GitHub是拒绝百度爬虫的，原因简单地来说，就是百度爬虫爬得太猛烈，已经对很多 Github 用户造成了可用性的问题了，详见这一篇文章。这个问题的解决思路是：

我们知道cdn能缓存静态资源，如果我们利用cdn制作我们的镜像网站，再将百度索引的解析cdn上，那么爬虫就不会访问github服务器了，而是访问cdn缓存服务器。

具体操作方法请见另外一篇文章，可操作性很强。

然而。遗憾的是，百度的抓取诊断一直是“抓取诊断”，仍未成功，一怒之下就把百度地图文件卸载了！。这就是那个彩蛋。。 :(
