R programming - manipulating data with dplyr

发表于 2017-06-02 | 分类于 Programming ， R |

Basic R commands on manipulating data with dplyr.

dplyr is a fast and powerful R package written by Hadley Wickham and Romain Francois that provides a consistent and concise grammar for manipulating tabular data. One unique aspect of dplyr is that the same set of tools allow you to work with tabular data from a variety of sources, including data frames, data tables, databases and multidimensional arrays.

Basic manipulating functions include select(), filter(), arrange(), summarize().

Foundation

select()

select a subset of columns

filter()

select a subset of rows

arrange()

make rows in order

summarize()

Summarize data profile, summarize() usually works together with group_by()

Supporting functions

mutate()

create a new variable based on the value of existed variables

quantile()

View()

See all records in scrip window, instead of 10 lines of preview in console window.
Note that the ‘V’ in View() is capitalized

%>%

%>% allows us to chain the function calls in a linear fashion. The code to the right of %>% operates on the result from the code to the left of %>%.

%>% 是链式操作操作符。%>%的功能是用于实现将一个函数的输出传递给下一个函数的第一个参数。注意这里的，传递给下一个函数的第一个参数，然后就不用写第一个参数了。
